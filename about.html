This is Hunter Osborn's senior project.

For my senior project, I decided to make a Full Stack Web Application.

I used HTML/CSS for my front end, PHP for my backend, and SQL for my database.

I also used the app XAMPP for hosting a local server and hosting the database.


This project is called BabyTrak, and it allows the user to:

  • Create an account
  • Enter information for the child
  • Track information about the child, such as feeding times, bedtimes, and even changing times
